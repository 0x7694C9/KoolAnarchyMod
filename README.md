# KoolAnarchyMod

This repository contains recovered and HEAVILY MODIFIED source code of an old plugin I made alongside someone else with the goal of banning people from Kaboom style servers that genuinely deserve it.

This is NOT a 1:1 replica of what it originally was, since the original plugin did not work. This has been heavily modified to work on 1.21.11 servers and actually do it's job of banning players and preventing players from joining if they are inside the `bans.yml` file.

A lot of this stuff uses my code from a few of my other plugins, cause you know what they say, "if it ain't broke, don't fix it".
